# The Conscious Family Centre — Website

A premium, multi-page website for The Conscious Family Centre, Abuja.

## Design Inspiration
- **Structure & Pattern:** Modeled after the Phenomenon Edu landing page (https://www.phenomenonedu.com.ng/edupreneur-challenge)
- **Brand Identity:** Green educational/family-centered palette inspired by the @consciousfamilycentre Instagram logo

## Files
- `index.html` — Landing page with hero, mission, programs, benefits, criteria, and CTA
- `about.html` — Story, mission/vision/values, and team
- `programs.html` — Detailed program descriptions
- `contact.html` — Contact form (Formspree-ready) + location info + map
- `style.css` — All styling for all pages
- `script.js` — IntersectionObserver animations, navbar, form handling

## Setup
1. Upload all files to your hosting (GitHub → Cloudflare Pages or Hostinger)
2. Replace `FORM_ID` in `contact.html` with your actual Formspree form ID
3. Update the Google Maps embed with the exact coordinates
4. Replace placeholder phone/email if needed

## Palette
- Deep Greens (#0a1f12 → #3ea862)
- Warm Gold Accent (#f0a82e)
- Clean White & Soft Gray cards
